Koszt projektu

  1. Budowa prototypu licznika, czujnika napełnienia oraz oprogramowania układowego (praca 4 pracowników + koszty komponentów =~15 000 zł
  2. Stworzenie i dostosowanie aplikacji mobilnej na systemy Android i iOS ~20 000zł
  3. Sprzęt serwerowy:
    • Serwer zarządzający (główny) + Failover = ~2 x 20 000zł (ostateczna cena zależy od uzyskanego rabatowania)
    • Serwer aplikacji + Failover = ~2 x 15 000zł (ostateczna cena zależy od uzyskanego rabatowania)
    • Serwer bazy danych SQL + Failover ~2 x 40 000zł (ostateczna cena zależy od uzyskanego rabatowania)
  4. Oprogramowanie serwerowe:
    • System operacyjny Windows 10 Embedded LTSC = ~6 x 500 zł
    • Microsoft SQL Always On = ~ 55 000zł (ostateczna cena zależy od uzyskanego rabatowania)
    • Stworzenie aplikacji serwerowej dla aplikacji mobilnej (back-end) = 150 000zł
  5. Instalacja i wdrożenie systemu = 1,5 - 2 miesięcy pracy 3-osobowego zespołu - ~50 000zł
